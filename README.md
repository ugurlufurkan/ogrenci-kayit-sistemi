# Öğrenci Kayıt Sistemi

## Proje Hakkında

Bu proje HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit bir öğrenci kayıt sistemidir.

## Özellikler

- Öğrenci ekleme
- Öğrenci düzenleme
- Öğrenci silme
- Öğrenci arama
- Aynı öğrenci numarasını engelleme
- LocalStorage ile verileri saklama
- Toplam öğrenci sayısını gösterme
- Boş liste durumunu gösterme

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- LocalStorage
